# BKS Mobile

## Project Overview

**BKS Mobile** is a mobile banking application developed for a rural bank (BPR).  
The application provides core mobile banking features for customers, along with **PPOB (Payment Point Online Bank)** services to support daily financial transactions.

This application is designed for bank customers and operates under internal banking and regulatory requirements.

---

## Core Features (High-Level)

- Mobile banking features for BPR customers  
- Account information and basic transaction services  
- **PPOB** features for bill payments and top-up services  
- Other supporting features related to banking operations  

Detailed transaction flows, financial rules, and security mechanisms are not exposed.

---

## Technical Stack

- **Mobile**: Flutter  
- **Backend**: PHP  
- **Architecture**: Modular structure with focus on maintainability  
- **API Communication**: REST API  

---

## Role & Responsibilities

My responsibilities in this project include:

- Developing **mobile application** features using Flutter  
- Developing **backend services** using PHP  
- Integrating application logic with **core banking systems**  
- Ensuring data consistency between mobile, backend, and core banking  
- Collaborating with related teams to support testing and release processes  

---

## Key Learnings

- Banking applications require high reliability and careful handling of data  
- Coordination between mobile, backend, and core banking is critical  
- Security and consistency are key in financial systems  
- Clear separation of concerns helps maintain complex banking applications
